# Koenigsegg Jesko Scrollytelling Showcase

This is a luxury car scrollytelling showcase built with [Next.js](https://nextjs.org), Framer Motion, and Tailwind CSS.

## Getting Started

If you have downloaded or cloned this repository, follow these steps to run the project locally on your machine:

### Prerequisites

Ensure you have **Node.js** installed. You can download it from [nodejs.org](https://nodejs.org).

### Installation and Setup

1. **Install dependencies:**  
   Open your terminal in the project directory and run:
   ```bash
   npm install
   ```

2. **Run the development server:**  
   Once the installation is complete, start the server by running:
   ```bash
   npm run dev
   ```

3. **View the website:**  
   Open your typical web browser and navigate to:
   ```
   http://localhost:3000
   ```

The application will load with all dynamic scrollytelling animations, interactions, and images!
